# AI引用协议

## 项目宗旨
本协议旨在构建一套知识共享许可框架，赋予创作者对AI系统的数据引用权限管理能力。在数字时代背景下，博客、视频等原创内容作为知识生产的重要形式，其创作者应享有自主决定作品是否被AI系统使用的根本权利。

## 协议体系

### 1. 全域开放协议（Full Access AI Citation Protocol, FACP）
**适用场景**：主动授权AI进行深度学习与内容生成的学术研究、开源项目等非商业用途  
**权限范围**：
- 允许AI系统完整访问文本、图像、音频等全部原始素材
- 支持模型训练、特征提取等深度处理
- 需注明原始来源及版本信息

### 2. 有限引用协议（Selective Access AI Citation Protocol, SACP）
**适用场景**：商业内容生产、专业领域知识库建设等需控制数据使用的场景  
**权限范围**：
- 允许AI获取预设元数据（metadata）：
  - 文本类：摘要、关键词、章节标题
  - 视频类：时间轴标注、场景描述、字幕文本
- 禁止直接复制原始内容
- 要求建立引用追溯机制

### 3. 禁用协议（No Access AI Citation Protocol, NACP）
**适用场景**：涉及敏感信息、商业机密或艺术家特别声明的内容  
**权限范围**：
- 禁止任何形式的数据抓取与处理
- 建议采用数字水印技术进行版权保护
- 违规使用将触发协议规定的追责机制

## 技术实现建议
1. **元数据标注系统**：通过标准化的JSON-LD格式嵌入协议声明
2. **访问控制层**：基于HTTP头部字段（`X-AI-Citation-Permission`）实现API级权限管理
3. **区块链存证**：使用IPFS+Filecoin构建分布式内容溯源网络

## 伦理准则
本协议遵循以下原则：
- **知情同意**：创作者需明确声明授权范围
- **最小必要**：AI系统仅获取完成任务所需的最少数据
- **可追溯性**：所有引用行为需保留完整日志
- **收益共享**：商业性使用需建立创作者收益分配机制
